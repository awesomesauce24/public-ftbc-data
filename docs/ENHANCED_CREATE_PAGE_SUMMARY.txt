### Post-Generation Menu (5 Options)
After wiki markup is generated, users see:

```
--- Actions ---
(1) Submit/Edit the page      - Check page status, create/update
```

## What Changed

### New Functions Added (150+ lines)

1. **`_handle_page_submission(object_name, markup, realm_name)`**
   - Checks if wiki page exists
   - Shows creation steps for new pages
   - Shows update instructions for existing pages
   - Returns: "created", "updated", or "cancelled"

2. **`_copy_to_clipboard(text)`**
   - Copies markup to Windows clipboard
   - Uses `clip.exe` for reliability
   - Graceful error handling
   - Returns: True/False success

### Enhanced Functions (50+ lines)

3. **`_create_object_interactive(obj, realm_name)` - UPDATED**
   - Now shows 5-option menu (was 3 options)
   - Shows "What next?" sub-menu after options 1/2
   - Returns choice: "another", "realm", or "exit"
   - Manages menu looping

4. **`cmd_create_page()` - UPDATED**
   - Handles return values from interactive function
   - Manages realm/object navigation
   - Maintains or reloads page status
