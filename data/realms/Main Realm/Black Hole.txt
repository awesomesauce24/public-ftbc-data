== Obtaining ==

To unlock Black Hole, you must collect all 5 code pieces and then enter the final code into the Factory keypad.

=== Code Piece Locations ===

Inside the chimney at Spawn.  
Code piece: 8

Go to the second house near Spawn.  
The piece is hidden just below the roof above the front door.  
Code piece: 1

Near [[X]] and the [[Yoyleland]] teleporter.  
Look under the map to find the third piece.  
Code piece: 6

Inside [[Remote]]'s room (where [[Revolver]] is located).  
Drop below to find this piece.  
Code piece: 4

Under one of the snow-covered trees in Snowy.  
Code piece: 9

=== Final Step ===

After collecting all 5 fragments, the full keypad code is:  
81649

Go to the side of the [[Factory]], interact with the keypad, and enter 81649 to unlock Black Hole.

== Description ==
Black Hole is a swirling dark sphere surrounded by a glowing ring, based on his BFDI appearance.
